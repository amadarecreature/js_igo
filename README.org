#+TITLE: JavaScript碁盤

* 概要

ブラウザ上で使える碁盤です。囲碁クエストのウェブ版に検討機能が無かったので作りました。

[[file:./screenshot.png]]

** 主な機能

- 盤面サイズは 1x1 〜 52x52 までの非正方形に対応
- 簡易的なSGFインポート/エクスポート
- 分岐記録・盤面表示機能
- コメント表示・設定機能
- 初期盤面でのフリー編集機能

SGFは次のプロパティに対応しています。
- ルートノードでの盤面サイズ指定(SZ)(非正方形サイズ可能)
- ルートノードでの先番指定(PL), 初期盤面指定(AB, AW, AE)
- ルートノードでの表示エリア指定(VW)
- 各ノードにつき1つのコメント(C)
- 着手(B, W)

囲碁クエストでエクスポートしたSGFをペーストして確認したほか、拾ってきた詰碁の棋譜をペーストして確認しています。
